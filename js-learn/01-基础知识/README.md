# 基础知识
## 1. 数据类型
在JavaScript中,有8中基本数据类型(7中原始类型和1中复杂类型)
- Number 数字类型
- String 字符串
- Boolean 布尔
- Undefined 未定义
- Null 
- Object
- BigInt
- Symbol

### typeof操作符
>因为ECMAScript的类型系统是动态的,所以需要一种手段来确定一个变量的类型。使用`typeof`操作符表示获取变量的类型。

对一个值使用`typeof`操作符会返回下列字符串值之一:
- `undefined` 表示值未定义
- `boolean` 
- `string`
- `number`
- `object` 表示为一个对象
- `function` 表示为函数
- `symbol` 表示值为符号
